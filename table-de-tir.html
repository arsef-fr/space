Table de Tir

La table de tir est un élément essentiel d'un bon décollage de fusée, elle permet de contrôler de manière sûre et précise l'allumage des dispositifs pyrotechniques tout en se trouvant à bonne distance pour des questions évidentes de sécurité.

Fonctionnalitées

Le mot d'ordre lors de la conception de cette table à été la sécurité, c'est pour cela quelle est équipée de plusieurs sécurité distinctes qui permettent d'eviter un déclenchement accidentel Pour cela, sont intégrées: 2 switch de mise hors tension de la sortie pyro, un lecteur de badge pour que seul la personne habilitée puisse effectuer le tir.

La table a 2 modes de fonctionnement: le premier est un mode manuel; lorsque le bouton rouge est pressé, la sortie est alimentée. Le second mode est un compte à rebours; lorsque le bouton rouge est pressé, il déclenche le compte à rebours précedement défini

Les nombreuses améliorations

La première version avait un inconvénient, elle devait etre relié physiquement à la fusée. Pas pratique pour pouvoir être situé à 100m de celle-ci, c'est pour cela que le point majeur de cette V2 a été de réaliser un boitier récepteur en communication radio avec la table de tir pour ne plus avoir de longs fils entre la table et la fusée

Les autres améliorations sont moins visibles: on est passé de 2 Arduino uno qui communiquaient via port serial à une unique carte Arduino Mega cela simplifie grandement le codage. De plus tout le cablage a été revue pour le simplifier et avoir quelque chose de plus propre (on s'y retrouvait plus du tout lol).
